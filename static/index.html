How It Works


Zoom Data is Captured


Using a custom account-level webhook app on the Zoom marketplace, meeting data is captured and processed as real-time changes occur.

Updates are Shared with Discord


When a Discord user initiates a "watch" on a meeting, any incoming Zoom data for that meeting is processed and shared with the Discord server in the form of an updating meeting status message.
